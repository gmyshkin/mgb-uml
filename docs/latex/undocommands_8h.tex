\doxysubsection{Detailed Description}
All changes to a \mbox{\hyperlink{classTikzDocument}{Tikz\+Document}} are done via subclasses of QUndo\+Command. When a controller (e.\+g. \mbox{\hyperlink{classTikzScene}{Tikz\+Scene}}) gets input from the user to change the document, it will push one of these commands onto the \mbox{\hyperlink{classTikzDocument}{Tikz\+Document}}\textquotesingle{}s undo stack, which automatically calls the redo() method of the command.
