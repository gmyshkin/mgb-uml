\doxysubsection{Detailed Description}
This class contains a tikz \mbox{\hyperlink{classGraph}{Graph}}, source code, file info, and undo stack. It serves as the model in the MVC triple (\mbox{\hyperlink{classTikzDocument}{Tikz\+Document}}, \mbox{\hyperlink{classTikzView}{Tikz\+View}}, \mbox{\hyperlink{classTikzScene}{Tikz\+Scene}}).
