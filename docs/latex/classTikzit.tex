\doxysubsection{Detailed Description}
\mbox{\hyperlink{classTikzit}{Tikzit}} is the top-\/level class which maintains the global application state. For convenience, it also holds an instance of the main menu for mac\+OS (or Ubuntu unity) style GUIs which only have one, application-\/level menu.
