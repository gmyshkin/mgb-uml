\doxysubsection{Detailed Description}
Display a \mbox{\hyperlink{classGraph}{Graph}}, and manage any user input that purely changes the view (e.\+g. Zoom). This serves as the view in the MVC (\mbox{\hyperlink{classTikzDocument}{Tikz\+Document}}, \mbox{\hyperlink{classTikzView}{Tikz\+View}}, \mbox{\hyperlink{classTikzScene}{Tikz\+Scene}}).
