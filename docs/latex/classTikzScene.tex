\doxysubsection{Detailed Description}
Manage the scene, which contains a single \mbox{\hyperlink{classGraph}{Graph}}, and respond to user input. This serves as the controller for the MVC (\mbox{\hyperlink{classTikzDocument}{Tikz\+Document}}, \mbox{\hyperlink{classTikzView}{Tikz\+View}}, \mbox{\hyperlink{classTikzScene}{Tikz\+Scene}}).
